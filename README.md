# Proyecto-Final-Elementos
Repositorio para hacer cambios del proyecto final del laboratorio de elementos
Pythonia RPG

Sistema de administración de aventureros RPG que corre completamente en consola, sin librerías externas ni interfaces gráficas.

Cómo correr el proyecto

1. Clonar el repositorio
2. Abrir en PyCharm
3. Ejecutar `Main.py`
## Archivos

## Archivos

- `Main.py` — menú principal · Carlo
- `datos.py` — lectura y escritura del archivo de datos · Carlo
- `registro.py` — registro y búsqueda · Felipe / eliminación de aventureros · Fabián
- `progresion.py` — entrenamiento, tienda e inventario · Felipe
- `combate.py` — batallas, estadísticas y rankings · Fabián
- ## Datos

Los aventureros se guardan en `aventureros.txt` con el siguiente formato:
nombre,clase,nivel,vida,ataque,defensa,oro,estado,item1|item2
## Equipo

- Carlo Ugalde arquitectura y datos
